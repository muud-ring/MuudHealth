## Advisories for Future Sprints

1. **MaterialPageRoute calls (8 remaining)** — 6 files still use imperative `Navigator.push(MaterialPageRoute(...))` for modal-style navigation. Target for full GoRouter ShellRoute migration in a future sprint.

2. **Local .withOpacity on dynamic colors (40+)** — Trends widget cards use `.withOpacity()` on function parameters and locally computed colors. These cannot be migrated to `.withValues(alpha:)` without broader refactoring since they operate on dynamic Color values, not static constants. Consider a lint rule to flag new `.withOpacity()` usage.

3. **Residual Color(0x...) literals (110)** — Semantic colors in data visualization (chart greens, stress reds, sleep blues) are not part of the design system. Consider extending MuudColors with a `data` or `chart` sub-palette in Phase 4 (Intelligence).

4. **Dart analyze in CI** — Static analysis could not run in the current sandbox. Add `flutter analyze` to CI pipeline as a gating check.
